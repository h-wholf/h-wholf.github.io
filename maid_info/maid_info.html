Que es termux-maid???

Es mi primer proyecto como estudiante autodidacta en el mundo de la programación y el scripting usando Bash Termux MAID es un explorador de archivos directamente en la Shell zsh o Bash en termux pudiendo ser usado directamente desde la terminal o agregarlo al escritorio mediante termux-witget

Uso:

Escribiendo en la terminal. El comando. maid sin importar el directorio en el que se encuentre Después de ejecutado el comando maid se mostrará en la interfaz la siguiente información:

1: número de archivo

2: nombre de archivo, en caso de ser un directorio se mostrará el mensaje "es un directorio "

3: tamaño de archivo o carpeta

4 : se muestran algunas opciones de los comandos aceptados por termux maid los demás comandos se explican a continuación

Comandos

a (atrás) :
regresa un directorio atrás desde el acuatual.
comando usado : cd ..

b (borrar) borra el archivo actual. :
comando usado : rm $archivo

c (entrar en el directorio o carpeta) : entra en la carpeta seleccionada.
comando usado: cd $archivo

o (Open) abre el archivo seleccionado. :
Dependiendo del tipo de archivo se abrirá con la app especificada en el sistema Android
ejemplo si es MP3 se abrirá con el reproductor de audio si es MP4 se abrirá con el reproductor de vídeo
Si es txt con algún editor de texto si es html con el navegador por defecto.
Comando usado: termux-open $archivo

p (promt) habré la Shell para usar el comando que se dese . :
Comando usado read promt $promt

v (vim) habré el archivo seleccionado para editarlo en vim .:
Comando usado : vim $archivo

x (exit) sale de termux-maid. :
Comando usado: exit
